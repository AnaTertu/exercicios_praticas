{

    lógica

}

{

    console.log(inicio);

}

for ( var inicio =0; inicio <= nomes.length - 1; inicio = inicio + 1)

{

    console.log(nomes[inicio]);

}
  • inicialização;- uma variável para contar as vezes que a repetição ocorreu.
  • verificação;- verifica se é para a estrutura continuar repetindo o código ou não.
  • atualização;- após fazer uma repetição, atualiza a variável utilizada na inicialização.
  • Lógicaa ser repetida em cada interação.
