Windows

  • - cd (comand director - cd / - leva ao repositório raiz - cd nomePasta - cd .. volta uma pasta)
  • - dir (lista pastas)
  • - mkdir (cria pasta)
  • - del / rmdir
  • cls (limpa)
  • del nomePasta (deleta todo arquivo que consta dentro deste diretório.)
  • rmdir nomePasta /S /Q (deleta o diretótio e todo arquivo dentro dele.)

Unix

  • - cd
  • - ls (lista pastas)
  • - mkdir (cria pasta)
  • - rm -rf
  • clear ou ctrl+l (limpa)
  • sudu su (permição)
  • rm -rf (deleta o diretótio e todo arquivo dentro dele.)
